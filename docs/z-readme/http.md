# http
- HTTP (HyperText Transfer Protocol)，即`超文本运输协议`，是实现网络通信的一种规范
- HTTP常被用于在`Web浏览器`和`网站服务器`之间传递信息，以明文方式发送内容，不提供任何方式的数据加密

# TCP
- 传输控制协议（TCP，Transmission Control Protocol）是一种面向连接的、可靠的、`基于字节流`的`传输层通信协议`
